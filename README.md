# Image Caption Generator Using Deep Learning
 This project implements an **Image Caption Generator** that automatically produces descriptive sentences for images using deep learning. 
 It combines **InceptionV3 (CNN)** for feature extraction and **LSTM** for sequence generation.

## Overview
- Dataset: Flickr8k  
- Frameworks: TensorFlow, Keras  
- Model: CNN (InceptionV3) + LSTM  
- Metrics: BLEU-1 to BLEU-4

 ## Technologies Used
- Python, TensorFlow, Keras  
- NumPy, Pandas, Matplotlib  
- Kaggle Notebook for training
